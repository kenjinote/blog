# eguiとは

rustで使えるGUIライブラリです。

# eguiを使ってみる

下記のコマンドで、eguiのサンプルを実行できます。

```
git clone https://github.com/emilk/eframe_template/ egui_test
cd egui_test
cargo run
```

あとはサンプルコードを触りながらいろいろ試せます。
