# コマンドプロンプトでPowerToysインストールする

PowerToysをコマンドプロンプトでインストールする方法を紹介します。

```
winget install Microsoft.PowerToys --source winget
```

# コマンドプロンプトでPowerToysアンインストールする

```
winget uninstall --name PowerToys 
```
