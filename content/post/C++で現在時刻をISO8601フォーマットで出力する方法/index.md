### 前提

- ISO C++ 20 標準 (/std:c++20)

### コード

```
#include <chrono>
#include <format>

std::string datetime = std::format("{:%FT%TZ}", system_clock::now());
```

以上、C++で現在時刻をISO8601フォーマットで出力する方法でした。
