# Windowsでパスの通った実行ファイルの場所を見つける方法

実行ファイルを指定してコマンドを実行するとき、その実行ファイルががどこにあるのかを知りたいときがあります。そんな時は下記のコマンドで実行ファイルの場所を調べることができます。

```powershell
where <実行ファイル名>
```

例えば、ペイント(mspaint.exe)の場所を知りたい場合は下記のようにします。

```powershell
where mspaint.exe
```
